Introduction to Object-Oriented Design

Design Fundamental, June 17, 2018

Object-Oriented Design is one of the fundamental principles of software development. Practicing these concepts can really enhance your future as a Software Developer.

Let's think about all the different types of candy that are out there. We have taffy, chocolate bars, hard candy, etc. Each of these types, have characteristics. some may be chewy, others are not chewable at all. If we think aobut these principles, we can apply that to our code.

Each candy type can be thought of as a class. Thoses classes are going to have properties that are common within them.

As an example our taffy class:

  • Flavor
  • Size
  • Chewable
Are all properties that can occur in this class.
